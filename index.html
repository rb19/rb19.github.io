Multi-Entry Bracket Generator

This is a bracket generator that allows multiple entries per person and generates the first round match ups.

This will take a CSV file containing player names and number of cars and return the pages with players.

This bracket generator is dedicated to the Lincoln Johnny Lightning Race Club (LJLRC), located in Lincoln, RI.

Instructions to use this bracket generator:

  1. Create a spreadsheet for each race class in Google Sheets.
    • The spreadsheet should only have two columns: Player names and Car count.
      • Download an example CSV file.
      • What the spreadsheet should look like.
  2. Export the spreadsheet from Google Sheets as a CSV file. Save the file to your device.
  3. Click the Import CSV button below to import a CSV file. Once imported, the brackets will appear at the bottom of the page.
  4. Inspect the brackets. Manually edit each cell as needed.
  5. Enter the name of the race class below.
  6. Print page to save the brackets.
    • On laptop: Hit Ctrl+P to print the page, and save the web page as a PDF.
    • On tablet: Hit the Share button in Chrome to save the web page as a PDF.
  7. Repeat steps 1~5 until all race classes have been saved as PDFs.
  8. Upload all the PDFs to a Google Drive folder for the month's race.
  9. Share the folder to the committee.
