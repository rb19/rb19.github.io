LJLRC Bracket Generator Test

This is a bracket generator that will take a CSV file containing player names and number of cars and return the pages with players.

Instructions to use this bracket generator:

  1. Set up a CSV file for a race class. The CSV should only have two columns: Player names and Car count. The first line of the CSV should be: Player name and Car count
  2. Click the button below to import the CSV file. This will run the code and return pages and their brackets.
  3. Enter the name of the race class below.
  4. Print or screenshot this page to save the brackets.
  5. Repeat until all race classes have been filled.
